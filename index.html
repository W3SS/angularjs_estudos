AngularJS custom directives

I am fascinated by AngularJS, which is bringing me some ease I got with Flex. But the learning curve is being a little tougher than I thought. So, instead of just learn and apply, I've decided to share my findings, in the hope to help others like myself. I will try to keep this as updated as I can.

So far I've created 3 directives, digging fragments of code from several sources, like Angular Discussion Group, AngularJS official documentation and Stack Overflow not to mention a few good blogs on the way.

I will not say what I did is the best way to do, but it is the best way I could do. So if you find any bug or something that would help make the code better, please let me know and feel free to fork me on GitHub.
